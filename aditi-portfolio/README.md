# Aditi Goyal — Portfolio

Netflix-themed AI PM portfolio with two views:
- **Recruiter** — metrics, projects, experience, skills
- **Explorer** — product stories, case studies, about

## Stack
Single-file HTML portfolio. No build step. Auto-deploys to Vercel on every push.

## Local preview
Just open `index.html` in any browser.

## Deploy
Connected to Vercel — any push to `main` triggers a live deploy.
